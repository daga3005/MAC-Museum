# React + TypeScript + Vite
## 🖼️ MAC Museum
Sitio web interactivo inspirado en un Museo de Arte Contemporáneo, diseñado para ofrecer una experiencia visual dinámica utilizando GSAP y Chakra UI.

🚀 Tecnologías utilizadas
Tecnología	Uso
React	Construcción de la interfaz
TypeScript	Tipado estático y mayor robustez
Vite	Entorno de desarrollo rápido
GSAP	Animaciones fluidas y profesionales
Chakra UI	Componentes accesibles y estilizados
HTML / CSS	Estructura y estilos base.

🎨 Características principales
Animaciones suaves y modernas con GSAP.

Diseño limpio y responsivo con Chakra UI.

Navegación intuitiva inspirada en museos reales.

Secciones dedicadas a obras, artistas y experiencias visuales.

Código modular y organizado con TypeScript.
